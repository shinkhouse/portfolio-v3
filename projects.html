Chatroom (Work in Progress)

As a summer sideproject, I made this chatroom application. So far, it allows a user to sign in or create a new user to access the chatrooms. There is currently a list of chatrooms that a user can choose from. The user selects a chatroom and can send a message to users in the room. The message gets sent to a database and returns the sent message along with an update of the chatroom. The chatroom updates periodically. My main focus with this project is becoming more familar with server-side technology, database management, and security. In the future, possible features may include adding a new chatroom, chatroom management, user administration, displaying active users, emoticons, image and video processing, and more.

Weather 2.0 Project (Work in progress)

This website allows you to view the weather via DarkSky's weather API and pulls the API through a proxy. Similar to my first attempt, this website allows you to get the seven day forecast. This new version allows you to get the current weather with detailed information, warnings and alerts, an hour-by-hour forecast, all from the location your in using Google Maps Geolocation APIs. In addition, there is now the capability to search for a location.
